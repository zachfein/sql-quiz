SQL Quiz

topics

SELECT columns; filter with WHERE, LIKE and IN; filter with inequalities; COUNT; SUM; AVG; INNER/LEFT JOIN; GROUP BY. More advanced problems include the combination of JOINs with aggregate functions, subqueries, etc.

further study concepts

SQLZoo for reinforcing concepts from the quiz; looking at object-oriented design and the many intricacies reading another person’s Python code.

requires

basic SQL

time

1-2 hours (lots of disparity in speed for this one)

authors

Joel Burton

Goals

Students will explore basic SQL in an command line environment that gives them instand feedback on when they have come up with the correct solution.

Gotchas

  1. Case sensitivity

  2. single vs double quotes

  3. for queries comparing to null, must use IS , not ‘=’

  4. semicolons…

Notes

  • Pairs differ quite dramatically on how long this takes, and it often doesn’t correspond with over technical level in the course so far. People who have experience with SQL zip through, often regardless of whether they’re doing well with the course material so far.

Code Reviews

Code Review 1

  • Students will have completed the SQL quiz. Review the answers.sql file in the exercise directory and ask about each of the ways that students responded. Ask about any attempts that didn’t work and discuss why they didn’t work.

  • Further study is just playing around with SQLZoo and checking out the source code for quiz.py to learn more about about the oo-design.

  • Try to enforce good use of case even though psql is case insensitive
